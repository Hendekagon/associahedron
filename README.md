# Associahedron

A little study of the associahedron,
based on the paper Realization of the Stasheﬀ polytope
by Jean-Louis Loday

## Usage

`(use 'associahedron.core)`

```
(psi '[2 1 3])
=> {:tree [[0 [1 2]] 3], :weight [2 1 3]}
```
